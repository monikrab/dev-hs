How to make a function in Haskell

To make a function in Haskell, you have two options:

- Named functions

- Lambda (anonymous) functions

A regular (named) function is written like so:

function arg1 arg2 argN = expression

A lambda function is written like so:

\arg -> expression

You can also write multi-parameter lambdas:

\arg1 arg2 argN -> expression
